# 📊 E-Commerce Customer Analytics & Business Insights

## 📌 Project Overview
This project focuses on analyzing real-world transactional data from an e-commerce platform to extract actionable business insights. By combining customer demographics with purchasing behavior, the analysis uncovers top-selling products, target audience segments, and geographical performance. This project demonstrates end-to-end data cleaning, data manipulation, and visualization using Python.

## 🛠️ Tech Stack & Libraries Used
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / VS Code

## 🧼 Data Pipeline & Cleaning Process
The dataset was sourced from Kaggle, consisting of two relational files (`basket_detail.csv` and `customer_details.csv`). The following steps were executed to prepare the data for analysis:
1. **Data Merging:** Combined relational tables using a common identifier via inner join operations.
2. **Handling Missing Values:** Identified null records across features and imputed them systematically.
3. **Deduplication:** Dropped repeated transaction records to maintain data integrity.
